# Sales Analysis Dashboard (Excel)

## Overview

This project presents an **interactive Sales Analysis Dashboard built using Microsoft Excel**.
The dashboard analyzes sales performance across different **categories, states, customers, and months**. It provides a clear visual representation of key business metrics such as **total sales, profit, and category performance**.

The goal of this project is to practice **data analysis, dashboard design, and interactive reporting using Excel**.

---

## Key Metrics

* **Total Sales:** $1.93M
* **Total Profit:** $247.96K

These KPIs help quickly understand the overall business performance.

---

## Dashboard Features

### Interactive Filters

The dashboard includes slicers that allow users to filter data dynamically:

* **Order Date (Month)**
* **Product Category**
* **State**

These filters help analyze sales data from different perspectives.

---

### Sales by Category

A chart showing sales performance across product categories:

* Furniture
* Office Supplies
* Technology

This visualization helps identify which category generates the highest revenue.

---

### Customer-wise Sales

A bar chart that highlights **top customers and their total purchases**, helping identify high-value customers.

---

### State-wise Sales Analysis

Displays sales performance across different **states**, helping identify regions with the highest revenue.

---

### Monthly Sales Trend

A line chart showing **sales trends across months (Jan–Dec)** to identify seasonal patterns and performance changes over time.

---

### Monthly Profit Trend

A line chart visualizing **profit trends across months**, helping analyze how profitability changes throughout the year.
