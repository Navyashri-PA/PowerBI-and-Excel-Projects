# HR Analysis Dashboard

## 📊 Project Overview

The **HR Analysis Dashboard** is a data visualization project designed to analyze employee-related data such as salary distribution, experience levels, hiring trends, and demographic insights.
The dashboard helps HR teams understand workforce patterns and make data-driven decisions.

## 🚀 Key Features

* Employee count analysis by **month and year**
* **Salary distribution by gender**
* **Average salary by experience band**
* **Top 5 countries with highest average salary**
* **Employee distribution by weight category**
* **Salary gap analysis between male and female employees**
* **Interactive filters** for age group, experience level, county, and salary band

## 📈 Dashboard Insights

The dashboard provides insights such as:

* Minimum salary in the dataset: **40K**
* Maximum salary in the dataset: **200K**
* Average employee experience: **9.71 years**
* Average last salary hike: **0.15**

It also visualizes:

* Hiring trends over time
* Experience-level distribution
* Gender-based salary comparison
* Geographic distribution of employees

## 🛠 Tools & Technologies Used

* **Power BI** – Dashboard creation and visualization
* **Data Cleaning & Transformation** – Power Query
* **Data Analysis** – DAX calculations

## 📂 Dataset

The dataset contains employee information such as:

* Employee ID
* Salary
* Gender
* Experience
* Age group
* Country
* Salary band
* Weight category

## 🎯 Purpose of the Project

The main objective of this project is to:

* Demonstrate **data analysis and visualization skills**
* Provide HR professionals with insights for **better workforce management**
* Identify **salary patterns and employee demographics**
