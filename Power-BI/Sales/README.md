# 📊 Sales Analysis Dashboard

An interactive **Sales Analysis Dashboard** designed to monitor and analyze business performance through key metrics, sales trends, and geographic insights. The dashboard provides a clear view of revenue, profit, costs, and customer behavior to support **data-driven decision making**.

---

## 🚀 Overview

The **Sales Analysis Dashboard** provides a comprehensive view of sales performance using interactive visualizations. It enables users to explore sales data across different dimensions such as:

- Order Category
- Country
- Product Category
- Year
- Customer Name
- Sales Manager

The dashboard helps businesses quickly understand their **financial performance and market trends**.

## 📈 Dashboard Features

### 1️⃣ Sales Trend by Month
Displays monthly sales performance, helping identify trends and seasonal patterns.

### 2️⃣ Sales by Customer
A treemap visualization showing how much each customer contributes to overall sales.

### 3️⃣ Sales by Country
A bar chart comparing sales performance across different countries.

### 4️⃣ Sales by Sales Manager
Compares the performance of different sales managers based on sales value.

### 5️⃣ Profit Contribution by Category
A waterfall chart showing how different product categories contribute to total profit.

### 6️⃣ Cost to Sales Ratio vs Profit
Scatter visualization used to analyze the relationship between cost, sales, and profit.

### 7️⃣ Order Details Table
Provides detailed order information including:
- Order ID
- Customer Name
- Order Value
